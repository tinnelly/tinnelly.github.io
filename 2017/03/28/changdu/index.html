猖都

彼得堡是个娼都。我这样说，肯定不准确，比起荷兰性都，这里的层次还差的很远。有名气的几家脱衣舞吧，价钱贵的惊人。倒是一些筒子楼一样的巷野，藏着斯拉夫少女的不可知的美丽。一车车的旅行团接踵而至，他们嘴上念叨着“扣女啦”，又不肯花大价钱，着立领polo衫牛仔裤阿迪鞋，下车就豚鼠般入洞寻欢。至于其中细节，金丝猫如何驯服黄种男，我是没有瞧见。这里的买卖，少了些可供传言的风月，多了些动物性的粗野。

那么就是个猖都，这么说起码准确一点。我的老乡在微信上向我吐槽自己公车被抢，损失了一个月的房租。据我所知，这些抢劫黑毛一般不会找女人下手。我和我班胖子一起上地铁，我看着他惨遭围困，上下其手。我毕竟贼城西安混过的，真金白银买来些许教训。可是这里，贼比我们还要羞怯，虽是团伙作案，一下就面红耳赤，不得手就像风一般逃离现场。胖子袒露被摸多次，没有一次让黑毛得手。他在离我半米的地方被几个黑毛围攻，那场面确实说不上来是行窃还是性骚扰，而且往往啥也没捞着，这就更加没法定性了。毛子蹩脚的偷窃技术一度令我们陷入尴尬。行窃是门技术活，硬生生让他们弄成体力活。这次女人被窃，可知国运是越来不济，黑毛这种二等公民也只能更加倒霉。最近又得知全城在闹反腐败的游行，我虽没什么感触，但我知道我们付着高昂的学费，系里却连模特都请不起。那个师大常驻老年模特，我把他从中年画到老。我怕我连做春梦，都是他的脸。

这么说，叫他个猖都不为过:）也许还有别的名字，比如齁都，那是我吃的第一碗红菜汤，就被齁住了，俄餐的文火配猛料，非常敢于下手。比如刚都：这里的一切都谢绝猥琐，只需要正面刚，游戏里深有体会。但这些都没有猖都上口。我允许自己黑自己的过往。毕竟同样的词汇，我不会再用第二遍。我不是些许经历，就吹一整年。一些人只道长安美，那是长安离得远。而且我们谁也没有见过长安，更是可以被吹上天。
