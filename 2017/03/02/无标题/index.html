田耐里

冷雪夜

冷夜面前，我得说，我对文本有处女情节。无论何种形式的白: 粗糙肌理的书写纸、车缝线的记事本、友好护眼的原木白、屏幕的背光白，都不能使我静止。我不担心写坏，逐步拥有更多的备用材料并未增添自己半分胆识与底气。我仍像初次发现书写秘密的洞穴人一样心潮澎湃。我发现，我只能像写代码那样创建文本，比如先写一个空语句，然后等待后续填补，线性在我这里被打乱，我称这种从四面八方录入文字的方法为：叠纪。

马克思·恩斯特显然聪明过我，我们窝在沙发床上上看录像，这个长得像第一个演吸血鬼的男人说 “我对白纸有处女情节”。然后掏出一个树皮开始拓印上面的纹路，就像幼时我们拓印一元硬币上的菊花一样。你看，这样就不是白纸了吧，而我也可以在上面创作。那些复杂的纹路此刻经过他的脑洞，创作变成填词游戏。我感觉脑浆喷射，抱着显示屏再也忍不住：哈哈哈，我知道了。

我并非想表达恩斯特后继有人，这早已不是新鲜事。但倘若文字可以拓印，我是否不那么畏惧白纸。幼时初次提笔，我就刻意在模仿一个成人，我将自己置身于整个文学史面前，惊涛骇浪汹涌而过，我像是晕水症患者，感觉自己正朝水流的方向倒去，于是真的栽进水里。我迷失在浩瀚的前人智慧前，这个庞大的集合体仿佛无限增生怪，而我在这里马失前蹄。更加遗憾的是，真正成年以后，我依然没有摆脱这种畏惧。我没有成为我所倾慕的人，事实上，我连这些人的影子都没捡到。他们恃才放旷，表情自在眼神放空，他们是有祖传秘方的流动摊点，错过之后，只能在无数失眠夜里一边默默吞口水，一边发誓，仅凭残存的味觉体验也要摸清用料明细。

我本可以抱着“我不写，就不会出错”的迷之优越走进温柔的良夜。如你所料，我陷入悲鸣自嘲，我的无趣与怯懦无需求证。我开始白日做梦，无论睡谁的被窝、领谁的便当，随时准备挂起，灵魂出逃。不得不承认，成人世界绝非易事，但唯独缺少一个独白者。我过着蝴蝶也无法梦见的平和日子，临近年末，被巨大的虎钳抓起，发现自己已身在“年终促销”的行囊。

然而雪夜没有喧嚣。这里只有刺骨的冷和永恒的静默。雪夜淹没声音和发声的一切。若你在雪夜里死去，没有人留意。身在雪夜，诡谲如我也无法制造幻象。我的手牌已打光，雪夜将我彻底看个仔细。我曾因拒绝真实而放浪形骸，而此刻我如何说出“若我不抛头露面，就没人知道我丑”。

羞耻愈合。我开始阅读曾不屑一顾教学指南，开始使用我曾经不耻的灰色颜料。省时省力不再是我所鄙夷的一项忠告。我明白生命有限，而历史之漫长，非我此生可以丈量。我所追逐的人，我本可列一个长长的名单，我与他们惜别。我审时度势，终于看清他们的轮廓，谁人不是握一手烂牌强装镇定，谁人不是凭一己之力对抗岁月洪荒。

趋近真实使人平静。神点燃他案头上的烛火，睡眼惺忪的人类踱步至花前月下，此刻我正是须臾寿命的雄蛾。焚烧过后，请捡我入酒。好吃，壮阳。
